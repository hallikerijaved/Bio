## ✨ Features

- ✅ **Fully Responsive Design** - Mobile, tablet, and desktop optimized
- ✅ **Smooth Animations** - AOS (Animate On Scroll) effects
- ✅ **Dynamic Text** - Typed.js for animated text sequences
- ✅ **Sticky Navigation** - Fixed header with smooth scrolling
- ✅ **Contact Form** - Integrated with Web3Forms for email submissions
- ✅ **Performance Optimized** - Minified CSS/JS, lazy loading images
- ✅ **SEO Friendly** - Semantic HTML5 structure
- ✅ **Cross-browser Compatible** - Works on all modern browsers

## 🚀 Quick Start

### Option 1: Open in Browser
Simply open `index.html` in your web browser.

### Option 2: Local Development Server
```bash
# Using Python 3
python -m http.server 8000

# Using Python 2
python -m SimpleHTTPServer 8000

# Using Node.js (if http-server is installed)
npx http-server

# Using PHP
php -S localhost:8000
```
Then navigate to `http://localhost:8000` in your browser.

## 🔧 Configuration

### Contact Form
The contact form uses **Web3Forms** for backend email delivery. To use it:
1. Sign up at [Web3Forms](https://web3forms.com/)
2. Get your API key
3. Update the form endpoint in `main.js`

### Customization
- **Colors**: Edit CSS variables in `assets/css/style.css`
- **Content**: Modify HTML sections in `index.html`
- **Images**: Replace images in `assets/img/`
- **Scripts**: Customize functionality in `assets/js/main.js`
