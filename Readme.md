# Javed Hallikeri Portfolio

This repository contains a static personal portfolio website for Javed Hallikeri.

## Tech Stack

- HTML5
- CSS3
- JavaScript
- Bootstrap
- AOS
- Typed.js
- Waypoints

## Project Structure

- `index.html` - main portfolio page
- `assets/css/style.css` - custom site styles
- `assets/js/main.js` - site interactions
- `assets/img/` - profile, hero, and favicon images
- `assets/vendor/` - only the vendor assets currently used by the page

## Run Locally

Open `index.html` in a browser, or serve the folder with any simple static server.

## Notes

- The contact form submits through Web3Forms.
- Unused template sections, vendor libraries, form PHP files, duplicate assets, and placeholder content have been removed.
